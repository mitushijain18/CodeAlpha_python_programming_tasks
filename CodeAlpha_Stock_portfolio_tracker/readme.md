# Stock Portfolio Tracker Using Python

## Project Overview

This project is developed as part of the CodeAlpha Python Programming Internship. The project focuses on creating a Stock Portfolio Tracker that helps users manage and analyze their stock investments.

The main objective of this project is to understand financial data management, user input handling, calculations, and Python programming concepts by building a simple application for tracking stock portfolios.

## About Stock Portfolio Tracker

A Stock Portfolio Tracker is an application that allows users to record their stock investments, monitor portfolio performance, and calculate important financial information such as total investment value, current value, and profit or loss.

This project provides a simple way to organize stock holdings and understand basic portfolio management concepts.

## Working of the Application

The Stock Portfolio Tracker works by allowing users to enter details about their investments, including stock name, purchase price, quantity, and current market price.

The program stores this information and performs calculations to display portfolio details. It calculates the total invested amount, current portfolio value, and overall profit or loss.

### Working Steps:

1. User enters stock details.
2. The program stores stock information.
3. The application calculates investment values.
4. Current stock prices are compared with purchase prices.
5. Profit or loss is calculated.
6. Portfolio summary is displayed.

## Calculation Features

The application performs calculations such as:

- Total Investment = Purchase Price × Quantity

- Current Value = Current Price × Quantity

- Profit/Loss = Current Value - Total Investment
